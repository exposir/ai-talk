# AI 开发工具汇总

业内最新 AI 编程助手与开发工具。

## 🏆 主流工具

| 工具               | 公司             | 特点                                        | 形态     |
| ------------------ | ---------------- | ------------------------------------------- | -------- |
| **Claude Code**    | Anthropic        | 终端 AI 助手，低错误率，擅长生产级代码      | CLI      |
| **Antigravity**    | Google           | Agent-First IDE，Gemini 驱动                | IDE      |
| **GitHub Copilot** | GitHub/Microsoft | 实时代码补全，多语言支持，生态最成熟        | IDE 插件 |
| **Cursor**         | Cursor           | AI 原生编辑器，Agent 模式可自动完成整个任务 | IDE      |
| **Windsurf**       | Codeium          | AI 增强 IDE，免费使用，无缝集成             | IDE      |
| **Kiro**           | AWS              | Spec-driven 开发，自动生成设计文档和任务    | IDE      |
| **Zed**            | Zed Industries   | Rust 构建，极速性能，开源，支持本地 LLM     | IDE      |
| **Trae**           | ByteDance        | SOLO 自主模式，Builder 模式可生成完整应用   | IDE      |
| **Devin**          | Cognition Labs   | 首个 AI 软件工程师，端到端自主开发          | Agent    |
