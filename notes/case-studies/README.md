# 工程案例研究

优秀项目的技术实现和工程经验学习。

## 案例列表

| 项目 | 说明 | 亮点 |
|------|------|------|
| [Telegram](./telegram/README) | 即时通讯应用 | 小团队、高性能、自研协议 |

## 选择标准

入选的案例需要满足：

1. **技术创新**：有独特的技术方案或优化
2. **工程效率**：团队规模小但产出大
3. **开源透明**：代码可研究学习
4. **行业影响**：对业界有启发意义

## 学习目标

- 理解优秀架构设计的思路
- 学习性能优化的方法论
- 掌握小团队高效协作的秘诀
- 应用到 LLM 应用开发中
