## Agent 框架对比

| 框架          | 特点             | 适用场景   |
| ------------- | ---------------- | ---------- |

| **LangGraph** | 状态管理，工作流 | 复杂 Agent |

| **CrewAI**    | 多 Agent 协作    | 团队模拟   |
