# Antigravity 概览

Google Antigravity 是 Google
DeepMind 于 2025 年 11 月推出的下一代 AI 编程平台，基于 VS Code 构建，采用
**Agent-First** 设计理念。

---

## 核心定位

> **从 AI 辅助 → AI 自主执行**

传统 AI 编程工具（如 Copilot）专注于代码补全，而 Antigravity 的 Agent 可以：

- 🧠 **理解** 自然语言需求和项目上下文
- 📋 **规划** 详细的实现方案
- ⚙️ **生成** 完整的多文件代码
- 🧪 **测试** 并自动调试
- 🔄 **迭代** 直到任务完成
- 📝 **记录** 生成 Artifacts 报告

---

## 核心组件

### 1. Editor View（编辑器视图）

基于 VS Code 的完整编辑器体验：

| 功能             | 说明                   |
| ---------------- | ---------------------- |
| **Tab 补全**     | AI 驱动的智能代码补全  |
| **自然语言命令** | 用自然语言描述代码意图 |
| **内联编辑**     | 执行过程中实时修正     |
| **代码 Diff**    | 清晰展示修改内容       |
| **多文件编辑**   | 跨文件协同修改         |

### 2. Agent Manager（代理管理器）

Agent 任务的控制中心：

| 功能              | 说明                |
| ----------------- | ------------------- |
| **任务创建**      | 创建和分配新任务    |
| **多 Agent 编排** | 多个 Agent 并行工作 |
| **进度监控**      | 实时查看执行状态    |
| **思维过程**      | 展示 Agent 的推理链 |
| **异步反馈**      | 执行中提供修正意见  |

### 3. 集成浏览器

Agent 可自动操作的浏览器：

| 功能           | 说明               |
| -------------- | ------------------ |
| **自动化测试** | UI 测试和功能验证  |
| **网页研究**   | 获取文档和参考资料 |
| **截图录屏**   | 记录操作过程       |
| **DOM 交互**   | 表单填写、点击操作 |

## 核心特性

### Multi-Agent 协作

多个 Agent 可以并行工作，各负其职：

```
Agent 1: 代码生成
Agent 2: 测试编写
Agent 3: 代码审查
Agent 4: 文档生成
```

### 知识库学习

Agent 可以：

- 保存有用的上下文和代码片段
- 从过去的任务中学习
- 在未来任务中应用经验

### 异步反馈系统

执行过程中提供修正：

- 类似 Google Docs 的评论功能
- 在截图和代码上添加注释
- Agent 实时调整计划

## 适用场景

### ✅ 最适合

| 场景         | 说明               |
| ------------ | ------------------ |
| 快速原型开发 | 从想法到可运行代码 |

### ⚠️ 注意事项

| 场景         | 建议             |
| ------------ | ---------------- |
| 大型遗留项目 | 分模块处理       |
| 高安全性项目 | 使用 Review 模式 |
| 离线工作     | 需要网络连接     |
| Beta 阶段    | 可能存在不稳定   |
